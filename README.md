## Player

### Registra jogador

```json
/api/players/register-player

{
  "name": "",
  "email": "",
  "telephone": "",
  "playerType": "AVENGERS"
}
```

### Listar jogador

```json
/api/players/list-player?page=0&size=1&sort=id
```
